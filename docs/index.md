# pysearch 文档

pysearch 是一个面向 Python 代码库的高性能、上下文感知搜索引擎，支持文本/正则/AST/语义搜索，提供 CLI 与可编程 API，适用于大型多模块项目的工程化检索。

特性
- 代码块匹配（函数/类/装饰器/导入/字符串/注释等）
- 上下文感知输出与结果排序
- 全项目索引与缓存，增量更新
- 多格式输出（text/json/highlight）
- CLI 与 API 双形态
